# прогноз зарплат в it ml и data science

это мой учебный проект по data science  
идея простая и практичная: не угадывать абстрактные цифры, а посмотреть, как может меняться зарплата по реальным вакансиям в it

я сделал приложение, которое само:
- забирает вакансии из hh api
- считает топ 5 популярных направлений
- обучает модель по каждому направлению отдельно
- строит прогноз зарплаты на несколько дней вперед
- показывает отчет и рекомендации обычным языком

проект не про “магическую ai кнопку”, а про понятный и рабочий пайплайн от данных до интерфейса

---

## что тут интересного

- прогнозируется не одна “средняя температура”, а отдельные направления
- есть автофильтр по популярности чтобы не выбирать все руками
- в интерфейсе сразу несколько графиков, по одному на каждое направление
- прогноз показывается как ожидаемая зарплата плюс минус диапазон
- если api временно не отвечает, проект умеет брать локальные csv

## источник данных

данные берутся через [hh api](https://api.hh.ru/)

по каждой вакансии используется:
- дата публикации
- название направления
- зарплатная вилка
- валюта

дальше зарплата приводится к рублям и агрегируется в дневной ряд

---

## как считается прогноз

для каждого направления делается свой временной ряд и свои признаки:
- lag 1 2 3 7
- скользящие средние
- скользящее стандартное отклонение
- день недели и выходной

потом:
1. ряд делится на train и test по времени без перемешивания
2. обучаются linear regression и random forest
3. выбирается лучшая модель по rmse и mae
4. строится рекурсивный прогноз на горизонт вперед
5. считается понятный диапазон прогноза плюс минус на основе ошибки модели

## как пользоваться

1. запускаешь `streamlit run app.py`
2. приложение само подгружает данные и обновляет отчет
3. смотришь топ 5 направлений по числу вакансий
4. по каждому направлению читаешь прогноз зарплаты и рекомендации
5. при необходимости нажимаешь кнопку обновления чтобы пересчитать все

## ограничения

- это учебный прогноз а не финансовая гарантия
- часть вакансий без зарплаты поэтому ряд может быть шумным
- конвертация валют упрощенная через фиксированные коэффициенты
- качество зависит от того сколько свежих вакансий есть в периоде
