# прогноз зарплат в it ml и data science

это мой учебный проект по data science  
я хотел сделать что то понятное и живое а не просто еще один скучный ноутбук

идея такая: приложение само забирает вакансии с hh, выбирает самые популярные направления, строит прогноз зарплаты и показывает это в виде отчета. 

---

## что умеет проект

- сам собирает данные из hh api
- хранит сырой и обработанный csv локально
- строит отдельный прогноз по каждому направлению
- выбирает топ 5 направлений по числу вакансий
- показывает графики и таблицу прогноза
- дает короткие рекомендации обычным языком
- умеет работать и через локальные данные если api временно не отвечает

## источник данных

данные берутся из [hh api](https://api.hh.ru/)

по каждой вакансии используются:
- дата публикации
- название направления
- зарплатная вилка
- валюта

потом зарплата переводится в рубли и собирается дневной ряд

---

## как устроен прогноз

для каждого направления строится свой временной ряд и набор простых признаков:
- lag 1 2 3 7
- среднее за 3 и 7 дней
- стандартное отклонение за 7 дней
- день недели
- выходной или нет

дальше все идет по обычной схеме
1. ряд делится на train и test по времени без перемешивания
2. обучаются `LinearRegression` и `RandomForestRegressor`
3. выбирается лучшая модель по rmse и mae
4. строится рекурсивный прогноз на 7 дней вперед
5. показывается диапазон прогноза плюс минус на основе ошибки модели

## как пользоваться

1. запускаешь `streamlit run app.py`
2. приложение само подгружает локальные данные или берет свежие если нажать обновление
3. смотришь топ 5 направлений по популярности

5. читаешь прогноз и короткую рекомендацию

## ограничения

- это учебный прогноз а не гарантия зарплаты
- часть вакансий без зарплаты поэтому ряд не идеальный
- конвертация валют упрощенная
- качество зависит от того сколько свежих вакансий есть в периоде
